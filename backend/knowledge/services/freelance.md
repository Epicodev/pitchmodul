# Epico Freelance

## Kort beskrivelse
Hurtig levering af erfarne freelance IT-konsulenter til konkrete projekter.

## Hvad vi leverer
- Erfarne IT-konsulenter på dag 1 til konkrete projekter og opgaver
- Senior-profiler med typisk +10 års erfaring
- Forhåndsscreenede kandidater fra eget netværk
- Konsulenter ansat hos Epico — kunden betaler en timepris

## Hvem servicen er til
- Virksomheder der har akut behov for IT-kompetencer på et konkret projekt
- IT-afdelinger der skal skalere op midlertidigt uden permanente ansættelser
- Kunder der ønsker fleksibilitet og ingen langtidsbinding

## Leveringsmodel
- Kunden sender request → Key Account Manager (KAM) modtager og sætter møde op
- Dedikeret Resource Manager starter søgning i globalt database
- Forhåndsscreenede CV'er sendes til kunde inden for typisk 48 timer (responstid)
- 3-vejs interview: kandidat, kunde, Epico
- Kontrakt underskrives — konsulenten starter
- Typisk fra request til kontrakt: 7-14 dage

## Nøgletal
- +600 konsulenter på kontrakt globalt
- +6000 CV'er i Epico's database
- 48 timers responstid på forhåndsscreenede CV'er
- 12 lande hvor Epico har konsulenter på kontrakt
- [VERIFICÉR] Gennemsnitlig kontraktlængde

## Typiske roller vi leverer
- Solution Architects, Software Architects
- Senior Developers (Java, .NET, JavaScript, Python, m.fl.)
- DevOps Engineers, Site Reliability Engineers
- Cloud Engineers (Azure, AWS, GCP)
- Cybersecurity-specialister
- Data Engineers, Database Administrators
- Project Managers, Program Managers, Scrum Masters
- Test Managers, Test Leads
- Integration Specialists
- ERP-konsulenter (SAP, Oracle, Dynamics)

## Stacks vi dækker
Bredt spektrum: Java, .NET (C#, VB), C/C++, Python, JavaScript/TypeScript, PHP, Cobol, Oracle, SQL Server, Azure, AWS, Kubernetes, Salesforce, SAP, ServiceNow, Citrix, m.fl.

## Hvad servicen IKKE er
- Vi headhunter ikke faste medarbejdere via Freelance — det hører under Epico Search
- Vi leverer ikke nyuddannede via Freelance — det hører under Epico NextGen
- Vi tilbyder ikke fastpris-projekter — vi sælger konsulenter på timepris/dagspris

## Prismodel
- Timepris baseret på rolle, seniority og marked
- Ingen rekrutteringsfee — kun timefee
- Ingen forpligtelse på minimum-volumen
- Kunden betaler kun mens konsulenten er på opgave
- [VERIFICÉR] Pris-eksempler eller -intervaller
