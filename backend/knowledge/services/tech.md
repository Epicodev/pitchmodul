# Epico Tech

## Kort beskrivelse
Specialister inden for Oracle Technology og Mainframe — to af de mest pressede niche-kompetencer på markedet. Også AI, AMS (Application Management Services) og Managed Technology.

## Hvad vi leverer

### Oracle-spor
- Dybe specialister i hele Oracle-stack (database, APEX, ERP, integration, cloud)
- 20+ års gennemsnitserfaring i Oracle-teamet
- Cutting-edge ekspertise specifikt i Oracle Database og Oracle APEX

### Mainframe-spor
- Eksperter til Mainframe-systemer (IBM z/OS, m.fl.)
- Cobol-udviklere og -arkitekter
- 25+ års gennemsnitserfaring i Mainframe-teamet
- Specialister i komplekse, intensive og forretningskritiske Mainframe-transaktioner
- Mainframe as a Service (operational drift og support)

### AI-spor
- AI Engineers / ML Engineers
- LLM Integration Specialists
- Data Scientists

### AMS / Managed Technology
- AMS Application Managers
- IT Operations Specialists
- Cloud Operations Engineers
- ITSM-specialister (ServiceNow, m.fl.)

## Hvem servicen er til

### Oracle:
- Virksomheder med tung Oracle-investering (database, ERP, integration)
- Organisationer der står overfor Oracle Cloud-migration
- Kunder der bygger applikationer på Oracle APEX

### Mainframe:
- Banker og forsikringsselskaber med Mainframe-core-systemer
- Transport- og logistik-virksomheder med historiske Mainframe-applikationer
- Virksomheder der er afhængige af Cobol-kode og kæmper med generationsskifte
- Organisationer der overvejer migration væk fra Mainframe — men vil vedligeholde imens

## Hvad servicen IKKE er

- Vi tager ikke fastpris-projekter
- Vi sælger ikke Oracle-licenser (kun konsulent-kompetencer)

## Hvorfor Tech er kritisk
- **Mainframe** specialister er en af de mest pressede kompetencer i DK
- Generationsskifte: mange Mainframe-eksperter går på pension i de kommende år
- Nye uddannelser leverer ikke Cobol-talenter — knowledge lever næsten kun i Epico's netværk
- **Oracle** kræver dyb teknisk specialisering — ikke noget generalister kan lære hurtigt
- Begge stacks er forretningskritiske hvor de bruges — fejl er meget dyre
